Adds Woocommerce product recommendation in a Widget located inside the CM Tooltip Glossary.

== Description ==

This plugin is an extension of the [CM Tooltip Glossary](http://wordpress.org/plugins/enhanced-tooltipglossary/).  You can use this CM Tooltip Glossary WooCommerce Widget to
display products from your Woocommerce shop inside a tooltip associated with any term or word appearing on your WordPress website and by this engage users to make content related purchases.
To associate terms to a Woocommerce product you need to edit the CM Glossary Tooltip term page and select the Woocommerce product id you want to associate to it. Both the prodcut name, product picture and a link to the product page will apear inside the tooltip once the user hover over the term

== Installation ==

1. Before instalation please download and install [CM Super ToolTip Glossary](http://wordpress.org/extend/plugins/enhanced-tooltipglossary/)

2. You need to have Woocommerce activated and working on your site

3. Upload the plugin folder to the `/wp-content/plugins/` directory

4. Activate the plugin through the 'Plugins' menu in WordPress
